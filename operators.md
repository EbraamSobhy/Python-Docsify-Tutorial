# <span style="color: #00D100">JavaScript Operators</span>

### - Javascript operators are used to perform different types of mathematical and logical computations.



## <span style="color:#00D100">Example 1</span>
```javascript
let a = 10;
let b = 3;

console.log(a + b);

console.log(a - b); 

console.log(a * b); 

console.log(a / b); 

console.log(a % b); 

console.log(a ** b);
```
